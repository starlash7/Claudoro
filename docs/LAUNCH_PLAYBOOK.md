# Claudoro Launch Playbook

## Distribution Strategy

Primary path is open-source usage from the repo:

```bash
git clone https://github.com/starlash7/Claudoro.git
cd Claudoro
npm install
npm run dev
```

Messaging rule:

- Avoid installer-first messaging.
- Lead with "open source developer focus timer".
- Ask users to star/watch the repo for weekly updates.

## Core Positioning

Developer focus timer with GitHub activity sync.

Long-form value statement:

Claudoro helps developers protect focus time and keep momentum visible by combining timer flow and GitHub activity context in one desktop app.
